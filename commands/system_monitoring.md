System Monitoring in Linux



Check Disk Space Usage
df -h

Explanation:
Dispalys disk space usage in a human readable format



Check Directory/File Size
du -sh 

Explanation:
Shows the total size of a file or directory in a human readable format



Check Memory Usage
free -h

Explanation:
Displays system memory usage RAM and swap



Check System Uptime and Load
uptime



Kernal and System Info
uname -a



Check and Manage System Services
systemctl status nginx
systemctl start nginx
systemctl stop nginx
systemctl restart nginx
systemctl enable nginx
